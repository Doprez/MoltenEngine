Flags Gets or sets the render flags for the current RenderCamera.
LayerMask Gets or sets the camera's layer render mask. Each enabled bit ignores a layer with the same ID as the bit's position. For example, setting bit 0 will skip rendering of layer 0 (the default layer).
MultiSampleLevel Gets or sets the multi-sample AntiAliasLevel when rendering a scene with the current RenderCamera.
Profiler Gets the RenderProfiler instance bound to the current RenderCamera, which tracks render performance and statistics of the scene rendered by the camera.
OrderDepth Gets or sets the ordering depth of the current RenderCamera. The default value is 0. Cameras which share the same output surface and order-depth will be rendered in the other they were added to the scene. If you intend to output multiple cameras to the same IRenderSurface2D, it is recommended you change the order depth accordingly.
