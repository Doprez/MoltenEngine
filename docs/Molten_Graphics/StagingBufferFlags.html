Read The buffer can be read from by the CPU via an internal staging buffer.
value__  
Write A staged vertex buffer can be written to by the CPU via an internal staging buffer. It is then copied to the main buffer. This is useful for data that changes less than once per frame. The GPU can read this type of buffer as fast as a default, but altering buffer data comes at a slight cost to performance.
