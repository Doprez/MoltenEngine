Properties  
Flags Gets the flags that were passed in when the texture was created.
DataFormat Gets the format of the texture.
IsBlockCompressed Gets whether or not the texture is using a supported block-compressed format.
Width Gets the width of the texture.
MipMapCount Gets the number of mip map levels in the texture.
ArraySize Gets the number of array slices in the texture.
MultiSampleLevel Gets the number of samples used when sampling the texture. Anything greater than 1 is considered as multi-sampled.
IsMultisampled Gets whether or not the texture is multisampled. This is true if is greater than 1.
Renderer Gets the renderer that the texture is bound to.
