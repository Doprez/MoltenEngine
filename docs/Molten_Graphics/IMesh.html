Molten.Graphics.IMesh

A base interface for mesh implementations.

Methods  
GetResource(UInt32)  
SetResource(IShaderResource, UInt32)  

Properties  
IsDynamic Gets whether or not the mesh was created as a dynamic mesh. Dynamic meshes are preferable when the mesh's data will be changing at least once or more per frame.
Material Gets or sets the material applied to the current mesh.
MaxVertices Gets the maximum number of vertices the mesh can contain.
Topology Gets the topology/structure of the mesh's data (e.g. line, triangles list/strip, etc).
VertexCount Gets the number of vertices stored in the mesh.
