Molten.Graphics.TextureFlags

Represents different options of a TextureAsset2D upon creation.

  Fields  
AllowMipMapGeneration Allows mip maps to be generated.
AllowUAV Allows unordered access to the texture.
Dynamic The texture is dynamic. Dynamic textures can only be written to by the CPU and read from by the GPU. The GPU cannot write (or copy to) a dynamic texture. Dynamic textures are best used in a situation where their data will be changed 1 or more times per frame by the CPU.
None  
NoShaderResource Do not create or expect a shader resource for the texture.
SharedResource Allow the resource to be shared between devices, which including devices of different DX versions.
Staging The texture is used purely for staging. Staging textures can only be read by the CPU and written to by the GPU. This is useful when you want to copy data off the GPU for reading by the CPU.
