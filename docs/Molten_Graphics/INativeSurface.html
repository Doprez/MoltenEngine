Molten.Graphics.INativeSurface

Represents a custom implementation of a native GUI control-based render surface.

Events  
OnHandleChanged Occurs when the underlying native has changed. Invoked by the renderer it is bound to.
OnParentChanged Occurs when the underlying, native control parent has changed. This affects the in some situations.
OnClose Invoked when the current has began it's closing process. Invoked by the renderer it is bound to.
OnMinimize Invoked when the current is minimized. Invoked by the renderer it is bound to.
OnRestore Invoked when the current is restored. Invoked by the renderer it is bound to.
OnFocusGained Invoked when the current gains focus.
OnFocusLost Invoked when the current loses focus.

Methods  

Properties  
Name Gets or sets the internal name of the underlying native control.
IsFocused Gets whether or not the current is focused.
Mode Gets or sets the window mode of the underlying control.
Handle Gets an to the handle of the underlying control.
ParentHandle Gets the current parent's control handle. Null if no parent is assigned.
WindowHandle Gets the handle of the window or form containing the current . This is not neccessarily it's direct parent of ancestor.
