Molten.Graphics.INativeSurface

Represents a custom implementation of a native GUI control-based render surface.

  Events  
OnHandleChanged Occurs when the underlying native Handle has changed. Invoked by the renderer it is bound to.
OnParentChanged Occurs when the underlying, native control parent has changed. This affects the WindowHandle in some situations.
OnClose Invoked when the current INativeSurface has began it's closing process. Invoked by the renderer it is bound to.
OnMinimize Invoked when the current INativeSurface is minimized. Invoked by the renderer it is bound to.
OnRestore Invoked when the current INativeSurface is restored. Invoked by the renderer it is bound to.
OnFocusGained Invoked when the current INativeSurface gains focus.
OnFocusLost Invoked when the current INativeSurface loses focus.

  Properties  
Name Gets or sets the internal name of the underlying native control.
IsFocused Gets whether or not the current INativeSurface is focused.
Mode Gets or sets the window mode of the underlying control.
Handle Gets an to the handle of the underlying control.
ParentHandle Gets the current INativeSurface parent's control handle. Null if no parent is assigned.
WindowHandle Gets the handle of the window or form containing the current INativeSurface. This is not neccessarily it's direct parent of ancestor.
