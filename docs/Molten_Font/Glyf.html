Molten.Font.Glyf

Glyf data table (glyf). This table contains information that describes the glyphs in the font in the TrueType outline format. Information regarding the rasterizer (scaler) refers to the TrueType rasterizer. See: https://docs.microsoft.com/en-us/typography/opentype/spec/glyf
