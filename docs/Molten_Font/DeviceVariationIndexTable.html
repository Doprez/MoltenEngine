Format  
DeviceStartSize The smallest size to correct, in ppem (pixels per em). Only set when Format is not VariationIndex.
DeviceEndSize The largest size to correct, in ppem (pixels per em). Only set when Format is not VariationIndex.
DeltaSetOuterIndex A delta-set outer index — used to select an item variation data subtable within an item variation store. Only set when Format is set to VariationIndex.
DeltaSetInnerIndex A delta-set inner index — used to select a delta-set row within an item variation data subtable. Only set when Format is set to VariationIndex.
DeltaValues An array of de-compressed delta values. Each element corresponds to a ppem size between DeviceStartSize and DeviceEndSize. The first array element (0) contains deltas for a ppem size of DeviceStartSize, while the last element contains deltas for a ppem size of DeviceEndSize. Only set when Format is not set to VariationIndex.
