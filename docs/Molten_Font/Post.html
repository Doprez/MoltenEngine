StandardOffsets Gets an array containing macintosh-standard gylph ID offsets. This is present in table version 2.5 when a font contains Macintosh-standard glyphs that have been reordered.
UnderlinePosition Gets the suggested distance of the top of the underline from the baseline, in font design units. Negative values indicate below baseline. The PostScript definition of this FontInfo dictionary key(the y coordinate of the center of the stroke) is not used for historical reasons. The value of the PostScript key may be calculated by subtracting half the underlineThickness from the value of this field.
UnderlineThickness Gets the suggested values for the underline thickness, in font design units.
