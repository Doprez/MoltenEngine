Molten.Font.PCLT

PCLT table - Printer Command Language Table The 'PCLT' table is strongly discouraged for OpenType™ fonts with TrueType outlines. Extra information on this table can be found in the HP PCL 5 Printer Language Technical Reference Manual available from Hewlett-Packard Boise Printer Division. See: https://docs.microsoft.com/en-us/typography/opentype/spec/pclt

FileName Gets a 6-byte field which is composed of 3 parts. The first 3 bytes are an industry standard typeface family string. The fourth byte is a treatment character, such as R, B, I. The last two characters are either zeroes for an unbound font or a two character mnemonic for a symbol set if symbol set found. See: https://docs.microsoft.com/en-us/typography/opentype/spec/pclt#treatment-flags
