Children Gets a collection containing all of the child SceneObject instances attached to the current SceneObject.
Components Gets a collection containing all of the SceneComponent objects attached to the current SceneObject.
Parent Gets the object's parent SceneObject, if any. Value is null if the current object has no parent.
