  Properties  
Audio Gets the main attached to the current instance.
Content Gets the main content manager bound to the current engine instance. It is recommended that you use this to load assets that are unlikely to be unloaded throughout the lifetime of the current session. You should use separate instances for level-specific or short-lifespan content. Disposing of a instance will unload all of the content that was loaded by it.
Current Gets the current instance of the engine. There can only be one active per application.
