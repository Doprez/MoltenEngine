Deadzone Gets or sets the stick deadzone as a percentage of the stick's total value range along each axis. The lower this value is set, the more sensitive the stick should be. However, setting it too low may cause the stick to become sensitive to unwanted movement.

Value Gets a containing the percentage value for each stick axis. The deadzone is taken into account when calculating this value.
X Gets the percentage value for the stick's X axis.
Y Gets the percentage value for the stick's Y axis.
